Although agile development divides software development into multiple iterations, it also requires that each iteration is a complete software development cycle, and regular process management must be carried out according to the methodology of software engineering.

Specifically, each iteration must complete the following five steps in turn.

  • Requirements analysis
  • Design
  • Coding
  • Testing
  • Deployment / evaluation

Each iteration lasts about 2-6 weeks.
